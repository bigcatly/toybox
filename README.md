# 玩具箱微信小程序

一个原生微信小程序版的“在线玩具箱”：不是电商集合页，而是把一组轻量、有趣、解压、可反复玩的互动小玩具放在一个统一入口里。

## 功能清单

- 首页：玩具搜索、分类筛选、今日推荐、随机开玩
- 敲木鱼：点击累计功德，带震动反馈和重置
- 决策转盘：自定义选项，随机帮你做选择
- 呼吸球：吸气、屏息、呼气的放松节奏引导
- 幸运骰子：掷骰并记录最近结果
- 电子泡泡纸：点破气泡解压，支持重置
- 今日抽签：抽取一句轻松好玩的签文

## 如何运行

1. 打开微信开发者工具。
2. 选择“导入项目”。
3. 项目目录选择本仓库根目录。
4. AppID 可以先用测试号，或把 `project.config.json` 里的 `appid` 替换成你的真实小程序 AppID。
5. 点击“编译”。
